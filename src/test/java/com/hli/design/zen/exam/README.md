# 设计模式学习验证考卷

> 🎯 通过编写代码验证你对23种设计模式的掌握程度

---

## 一、考卷说明

本目录包含23种设计模式的**学习验证测试类**，类似于编程考试：
- 每个测试类都有 `TODO` 标记，需要你补充代码
- 运行测试验证你的实现是否正确
- 全部测试通过 = 学习成果合格 ✅

## 四、使用方法

### 4.1 运行单个考卷
```bash
mvn test -Dtest=SingletonExam
```

### 4.2 运行所有考卷
```bash
mvn test -Dtest=*Exam
```

### 4.3 查看测试报告
```bash
mvn test -Dtest=*Exam -Dsurefire.reportFormat=brief
```
